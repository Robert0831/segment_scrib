## 使用 pipreqs 
去取得當前專案的環境

```shell
pip install pipreqs
pipreqs ./ --encoding=utf8
```




# 直接下載(包含weight和環境)

取得conda環境
```shell
#Create fold for the env
\Users\username\anaconda3\envs\'[env name]'
cd C:\Users\username\anaconda3\envs\'[env name]'

# 取得環境
tar -xzf env.tar.gz 
```
